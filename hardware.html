Parts of a Computer

A motherboard is a printed circuit board that allows the CPU, RAM, and all the other computer hardware components to communicate with each other. The CPU is the Central Processing Unit is the part of a computer system that is commonly referred to as the "brain" of the computer. The RAM is the random access memory and you can access any memory cell directly if you know the role and column that intersects cells. The hard drive is a data storage device used for storing and retrieving digital information using one or more rigid rapidly rotating disks coated with magnetic material.
